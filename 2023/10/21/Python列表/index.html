Python列表

列表的常用操作

  列表的作用是一次性存储多个数据，程序员可以对这些数据进行的操作有：增、删、改、查。

下标

函数

  • index()：返回指定数据所在位置的下标。

  • 语法：

    列表序列.index(数据,开始位置下标,结束位置下表)

如果查找的数据不存在则报错

  • count()：统计指定数据在当前列表中出现的次数。
  • len()：访问列表的长度，及列表中的数据的个数。

in: 判断指定数据在某个列表序列,如果在返回True,否则返回False

  • not in：判断指定数据不在某个列表序列，如果不在则返回True，否则返回False

案例：查找用户输入的名字是否已经存在

增加

作用：增加指定数据到列表中

  • append()：==列表结尾追加数据，加的是整体==

    语法：列表序列.append(数据)

  • extend()：==列表结尾追加数据，如果数据是一个序列，则将这个序列的数据逐一添加到列表，加的是拆开的.==

    语法：列表序列.extend(数据)

](https://img-1320281775.cos.ap-beijing.myqcloud.com/202309171553775.png)

  • insert()：指定位置插入

  • del：删除数据，可指定下标，默认删除整个

    语法：del 列表序列[下标]

  • pop()：删除指定下标的数据默认删除最后一个数据 ，可指定，有返回值（返回被删的数据）。

  • remove()：移除列表中某个数据的第一个匹配项。

    语法：列表序列.remove(下标)

  • clear()：清空列表

    语法：列表序列.clear()

修改

修改指定下标数据

  • 逆置：reserve()

  • 排序：sort()

    语法：列表序列.sort(key = None, reverse = False)

注意：reverse表示排序规则，reverse = True降序，reverse = False升序
