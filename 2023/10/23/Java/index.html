Java

多态

多态的概念

  多态是方法或对象具有多种形态，是面向对象的第三大特征。
多态的前提是两个对象（类）存在继承关系，多态是建立在封装和继承基础之上的。

多态的具体体现

  对象的多态是多态的核心和重点。

规则：

  一个对象的编译类型与运行类型可以不一致
  编译类型在定义对象时，就确定了，不能改变，而运行类型是可以变化的
  编译类型看定义对象时 = 号的左边，运行类型看 = 号的右边

对象类的转换

向上转型

向下转型

二者区别：

向上转型:通过子类对象(小范围)实例化父类对象(大范围)，这种属于自动转换

失去了子类新增的方法，只能调用父类里面已经有的方法

向下转型:通过父类对象(大范围)实例化子类对象(小范围) .这种属于强制转换

  在Java中，向下转型则是为了.通过父类强制转换为子类.从而来调用子类独有的方法,为了保证向下转型的顺利完成。在Java中提供了一个关键字instanceof ,通过instanceof可以判断某对象是否是某类的实例,如果是则返回true, 否则为false

instanceof

  关键字instanceof，判断一个对象是否为某个类（或接口）的实例或者子类的实例。

语法：

​ 对象（或者对象引用变量）instanceof 类（或接口）

Object类

  • Object类是所有类的父类，每个类都直接或间接继承Object类。
  • 如果没有显式地指定父类，那么该类会默认继承Object类。

Object类常用方法：

方法名称 方法说明
boolean equals() 判断两个对象是否“相等”
int hashCode() 返回对象的哈希码值
String toString() 返回对象的字符串表示形式
